Hello World Directive

AngularJS directive allow creating custom HTML elements, extend the functionality of existing HTML elements, and create self contained user interface components. This example illustrates creating the simplest directive. Let's create a <hello-world> directive which just says "Hello World". The "Hello World" message below is created by the <hello-world> element.

Name: {{customer.name}}
Street: {{customer.street}}

This App

This application declares a module called HelloWorldDirective. It depends on two other modules which happen to be some directives used to document these examples.
